## Descripción
Este proyecto muestra un programa básico en Java que imprime información personal y la fecha actual del sistema.

## Pasos de instalación
1. Instalar JDK 
2. Configurar el IDE (se usó VS Code)
3. Crear archivo `HolaMundoNF.java`
4. Compilar con `javac HolaMundoNF.java`
5. Ejecutar con `java HolaMundoNF`
